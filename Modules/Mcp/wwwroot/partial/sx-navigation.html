Tabs JS Plugin
Home pane. This content is visible by default.
Profile pane. Hidden until the Profile tab is clicked.
Contact pane. Each pane fades in via the .fade class on .tab-pane.
Disabled pane. Unreachable via the disabled tab button.
Pills JS Plugin
Home. Pills use data-bs-toggle="pill" instead of "tab" — same JS plugin, different style.
Profile. The active pill gets the .active class automatically.
Contact. Works identically to tabs under the hood.
Vertical Pills JS Plugin
Home. Vertical pills use aria-orientation="vertical" on the nav container.
Profile. Good for sidebar navigation patterns.
Messages. The content panel grows to fill available space via flex-grow-1.
Settings. Keyboard navigation works out of the box — arrow keys move between tabs.
